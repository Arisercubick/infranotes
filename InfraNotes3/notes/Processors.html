Processors is the brain of the computer, controlling all
of the components of a computer and executing their functions

Top companies that produces these chips are Intel and AMD with other manufacturers existing but not as dominant compared to these two behemonths

CPU Architecture

  • Control Unit: Feeds input and produces outputs of the cpu
  • Arthmetic Logic Unit (ALU): Produces the arthmetic and logical operations of the cpu
  • Registers: Small and highspeed storage which the cpu can access quickly as it is inbuilt within the CPU for the ALU to access
  • Main Memory Unit: It is the place where the cpu can hold temporary or permanent memory (Through either the memory or storage respectively)

Arithmetic Logic Unit (ALU)

Link to the ALU page
  • It is the key part of the CPU (Central Processing Unit
  • It performs all the arithmetic and logical operations that a computer needs to process data

Main memory unit (Random Access Memory)

  • Random access memory is the main memory of a computer where data and programs are temporaily stored while the cpu using them
  • Temporary access: holds data and while the computer is on
  • Fast Access: The CPU can read and write data from RAM much faster than from a hard drive or SSD which is a permanent storage
  • Random Access: The CPU can go directly to any memory without needing to read the previous address

Registers:

  • Small
  • High speed storage located inside the CPU used to hold data, instructions and addresses

Processor Speed

  • Speeds are measured in gigahertz (GHz)
  • 1 Hertz = 1 clock cycle per second
  • Most processors are near 5 GHz

Cache

Link to memory and storage for more information
  • A small, very fast memory inside or close to the cpu

Levels of Cache

L1
Fastest, smallest in size
L2
Very Fast, medium in size
L3
Slowest, largest in size

Multi Core processors

Each core can fetch, read, execute and decode it's own instructions

  • Dual-core processors combine 2 CPUs in a single unit.
  • Tri-core processors combine 3 CPUs in a single unit.
  • Quad-core processors combine 4 CPUs in a single unit.
  • Hexa-core processors combine 6 CPUs in a single unit.
  • Octa-core processors combine 8 CPUs in a single unit.
  • Duodec-core processors combine 12 CPUs in a single unit.
  • Hexadeca-core processors combine 16 CPUs in a single unit.

LLC - Last Level Cache (Level 3) - typically shared by all the cores

Memory controller

acts as the intermediary between the CPU and
main memory, managing all data transfers,
ensuring accurate and efficient reads and writes

Pipeliningis a technique where the CPU overlaps
the execution of multiple instructionsto improve
performance, rather than executing one

CPU Cooler

  • Computers are based off of microsopuc transistors.
  • Every time one of these transistors switches from one to another, it generates a small amount of heat
  • Multiply this small amount of heat over billions of transistors switching billions of times
    every second, you wind up with an amount of heat
    that you need to actively manage.
  • You can use air cooling, liquid cooling or passing cooling (a heat sink)
