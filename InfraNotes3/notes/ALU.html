ALU (Arithmetc Logic Unit)

The ALU is a key part of a cpu (Central Processing Unit)


It performes all the arithmetic and logical operations tat a computer needs
to process data

Random Access Memory

  • It's the main memory of a computer where data and programs are temporarily
    stored while the CPU is using them.
  • Temporary: holds data only while the computer is on. Everything is lost when the power is turned off.
    (That's why it's called volatile memory).
  • Fast Access: The CPU can read and write data from RAm much faster than from a hard drive or SSD.
  • Random Access: The CPU can go directly to any memory location without reading all the previous ones

CPU instructions

A CPU instruction have two parts to it, the Opcode and Operands

Opcode (operation code)
To either perform an operation like a calculation and load/create/store/read information
It basically the guidelines for the cpu to follow to do with the data
Operands (Specify where the data to be used)
Show where is the information is stored or located
Basically the address to direct the cpu to do its intended instruction

CPu instruction cycle

  1. Fetch: An instruction is read from the memory
  2. Decode: The retrieved instruction, the operation, abd the necessary addresses/data are all decoded and determined
  3. Execyte: The ALU performs the operation according what is specified in the instruction
  4. Store: The results of the operation are stored back into memory when indicated
