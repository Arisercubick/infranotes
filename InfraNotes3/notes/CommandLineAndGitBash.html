File System

How the system organizes information

Commands

Command What it does In the Library Analogy
cd (directory) Change directory Explore different shelfs in the library
ls List contents List the books in the shelf
mkdir (directory) Make a new folder Create a new shelf
rm (file) Remove a file Throw away a book
rm -r (folder) Remove a folder Remove a whole shelf with all books
my (source) Move a book to a different shelf or rename its title Move a book to a different shelf or rename its title
