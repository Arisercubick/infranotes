2025-10-20: Memory and Storage

Memory and storage are one of the main components of a computer being able to function

Memory

  • Memory is used when programs are running
  • Temporary Storage, meaning if the power is cut, it will turn off
  • Faster compared to the storage
  • Volatile, meaning it turns off and deletes data when the power is off
  • It is generally smaller than the storage
  • It is used for immediate and short term needs

Kinds of Memory

It is an umbrella term which contains:

  • Cache
  • Main Memory (RAM)
  • Read Only Memoruy (ROM)

Memory is also a trade off, you either want Speed or Capcity due to the cose

(Bandwith Increases) (CPU) -> Cache Level1 -> Cache Level2 -> Cache Level3 -> RAM Memory -> Mass Storage (Latency and Size increases)

Storage

  • Include documents, pictures, videos and operating system file
  • Long term memory of the computer
  • Non-Volatile, meaning it stores data even when the power is off
  • Slower compared to the memory of the devices
  • It is generally larger than the storage
  • It is used for long term needs

Levels of Cache

L1
Fastest, smallest in size, usually most expensive
L2
Very Fast, medium in size
L3
Slowest, largest in size
Ln
Can go on constantly depending on architecture
